# Entity
Корневой абстрактный класс для всех существ и объектов существующих в симуляции.
У каждой сущности и обьекта есть параметр проходимости
## TerrainObject 
Абстрактный класс для объектов которые просто занимают клетку. 
Не могут совершать действия/нельзя совершать над ними
### Rock
### Tree
## Supply 
Абстрактный класс для объектов которые являются ресурсом для других существ. 
Не могут совершать действия, но с ними могут взаимодействовать другие существа
### Grass
## Creature 
Абстрактный класс для существ которые могут передвигаться по карте
- Количество хп
- Цель существа(к какому обьекту на карте он производит движение)
### Herbivore
Травоядное, стремятся найти ресурс (траву), может потратить свой ход на движение в сторону травы, либо на её поглощение.
### Predator
Хищник. в дополнение к полям класса Creature, имеет силу атаки. На что может потратить ход хищник:
- Переместиться (чтобы приблизиться к жертве - травоядному)
- Атаковать травоядное. При этом количество HP травоядного уменьшается на силу атаки хищника.
## Nothing 
Пустая клетка. Единственная реализация Entity через обьекты которой можно проходить
