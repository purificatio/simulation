# Entity - Корневой абстрактный класс для всех существ и объектов существующих в симуляции.
    - у каждой сущности и обьекта есть параметр проходимости
    # TerrainObject - абстрактный класс для объектов которые просто занимают клетку. 
            Не могут совершать действия/нельзя совершать над ними
        ## Rock
        ## Tree
    # Supply - абстрактный класс для объектов которые являются ресурсом для других существ. 
            Не могут совершать действия, но с ними могут взаимодействовать другие существа
        ## Grass
    # Creature - абстрактный класс для существ которые могут передвигаться по карте
        - Количество хп
        - Цель существа(к какому обьекту на карте он производит движение)
        ## Herbivore - Травоядное Стремятся найти ресурс (траву), может потратить свой ход на движение в сторону травы, 
            либо на её поглощение.
        ## Predator - Хищник В дополнение к полям класса Creature, имеет силу атаки. На что может потратить ход хищник:
            Переместиться (чтобы приблизиться к жертве - травоядному)
            Атаковать травоядное. При этом количество HP травоядного уменьшается на силу атаки хищника.
    ## Nothing - пустая клетка. Единственная реализация Entity через обьекты которой можно проходить

## GameMap - хранение существ и их расположения на карте.
## MapGeneration - первоначальная генерация карты
## EntityFactory - создание существ
# Renderer - отрисовка карты
    ## ConsoleRenderer - реализация отрисовки в консоль
        ## Sprites - enum со спрайтами(эмоджи) существ в консоли

# Simulation - состояние игры
    InitActions - действия которые выполняються перед стартом симуляции
    TurnActions - действия которые выполняються каждый ход
